# Vuln Scan 漏洞扫描工具

## 1. 项目概述

Vuln Scan 是一款功能强大的漏洞扫描工具，专为网络安全专业人员和渗透测试人员设计。该工具提供了直观的图形界面，支持多种漏洞检测、AI分析、消息推送等功能，帮助用户快速发现和分析目标系统中的安全漏洞。

## 2. 功能特性

### 2.1 漏洞扫描
- **支持的漏洞**：
  - CVE-2022-22963（Spring Cloud Gateway 远程代码执行漏洞示例）
  - QVD-2026-14149（漏洞示例）
- **扫描方式**：
  - 快速扫描：针对单个目标进行扫描

### 2.2 网络配置
- **代理设置**：
  - 支持 HTTP 和 SOCKS5 代理
  - 可配置代理主机、端口、用户名和密码
- **Dnslog 设置**：
  - 集成 CEYE 服务

### 2.3 消息推送
- **支持的平台**：

  - 钉钉

- **推送内容**：
  - 漏洞扫描结果

- **分析功能**：
  - 漏洞分析

### 2.5 日志管理
- **日志控制**：
  - 日志总开关

### 3.2 核心模块

- **功能**：将扫描结果推送到指定平台

- **功能**：管理工具的各项配置

## 4. 使用指南

### 4.1 基本操作
1. **启动工具**：运行 `vuln-scan.exe`
2. **输入目标**：在顶部输入框中输入目标 URL
3. **选择漏洞**：从下拉菜单中选择要检测的漏洞类型
4. **开始扫描**：点击"开始扫描"按钮
5. **查看结果**：在中央区域查看扫描结果和日志

### 4.2 高级设置

#### 4.2.1 代理设置
1. 点击"设置菜单" → "代理设置"
2. 启用代理并配置代理服务器信息
3. 点击"保存"按钮

#### 4.2.2 Dnslog 设置
1. 点击"设置菜单" → "CEYE设置"
2. 启用 CEYE 并配置 Token 和 Domain
3. 点击"保存"按钮

#### 4.2.3 消息推送设置
1. 点击"设置菜单" → "消息推送"
2. 启用消息推送并选择平台
3. 配置推送 URL 和相关参数
4. 点击"保存"按钮

#### 4.2.4 AI 分析设置
1. 点击"设置菜单" → "AI 设置"
2. 启用 AI 分析并选择 AI 提供商
3. 配置 API Key 和相关参数
4. 点击"保存"按钮

#### 4.2.5 日志设置
1. 点击"设置菜单" → "日志设置"
2. 使用滑块控制日志输出选项
3. 点击"保存"按钮

### 4.3 批量扫描
1. 点击"操作菜单" → "打开文件"
2. 选择包含目标列表的文件
3. 选择漏洞类型
4. 点击"开始扫描"按钮

## 5. 技术实现

### 5.1 漏洞检测原理
- **CVE-2022-22963**：利用 Spring Cloud Gateway 的 SpEL 表达式注入漏洞示例
- **QVD-2026-14149**：利用特定系统的远程代码执行漏洞示例
- **检测流程**：
  1. 构造漏洞检测请求
  2. 发送请求到目标服务器
  3. 分析响应结果
  4. 确认漏洞是否存在

### 5.2 AI 分析实现
- **分析流程**：
  1. 收集漏洞信息和上下文
  2. 构建 AI 分析请求
  3. 调用 AI API 获取分析结果
  4. 解析和展示分析结果

### 5.3 消息推送实现
- **推送流程**：
  1. 收集扫描结果和漏洞信息
  2. 构建推送消息
  3. 根据平台类型发送推送
  4. 处理推送结果

### 5.4 GUI 实现
- **技术**：使用 Fyne 框架构建跨平台 GUI
- **布局**：采用 Border 布局，分为顶部、中央和底部三个区域
- **主题**：自定义深色主题，提供良好的视觉体验

## 8. 安全注意事项

1. **使用责任**：本工具仅用于合法的安全测试和授权的渗透测试
2. **目标选择**：确保只对授权的目标进行扫描
3. **网络影响**：某些漏洞扫描可能会对目标系统造成影响，请谨慎使用
4. **API 密钥**：妥善保管 AI 服务的 API 密钥，避免泄露
5. **配置安全**：配置文件中可能包含敏感信息，请妥善保管
